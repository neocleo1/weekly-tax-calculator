# Weekly Tax Calculator

A Java program that calculates weekly tax withholding based on income brackets.

## Description

This program prompts users to enter their weekly income and calculates the appropriate tax withholding amount based on progressive tax brackets. It demonstrates the use of if-else-if selection structures in Java.

## Tax Brackets

- Income less than $500: 10% tax rate
- Income $500 to $1,499: 15% tax rate
- Income $1,500 to $2,499: 20% tax rate
- Income $2,500 and above: 30% tax rate

## How to Run

1. Clone the repository
2. Open the project in Eclipse IDE or any Java IDE
3. Run the `Calc.java` file
4. Enter your weekly income when prompted
5. View your calculated tax withholding

## Example

```
Enter your weekly income: 
1250
Weekly Income: 1250.0
Tax withholding: 187.5
```

## Requirements

- Java SE 25 or higher
- Eclipse IDE (or any Java IDE)
